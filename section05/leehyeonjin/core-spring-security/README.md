## 해당 실습코드 목표

### 스프링 시큐리티 인가 개요

> 인가(Authorization) 프로세스 구현 - DB 연동
> - DB와 연동하여 자원 및 권한을 설정하고 제어함으로 동적 권한 관리가 가능하도록 한다.
> - 설정 클래스 소스에서 권한 관련 코드 모두 제거.
> - 관리자 시스템 구축
>   - 회원 관리 : 권한 부여.
>   - 권한 관리 : 권한 생성, 삭제.
>   - 자원 관리 : 자원 생성, 삭제, 수정, 권한 맵핑.
> - 권한 계층 구현
>   - url : url 요청시 인가 처리.
>   - method : method 호출시 인가 처리(method, pointcut).
